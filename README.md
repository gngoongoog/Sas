# 🚀 منظومة SuperSAS v4 - نظام الإدارة الفائق لمشتركي مايكروتك وراديوس

### 💻 لوحة التحكم الاحترافية والذكية لإدارة شبكات الإنترنت والاشتراكات لشبكات MikroTik و Radius

يسرنا تقديم **SuperSAS v4**، السلسلة الرابعة والأنضج من نظام الإدارة الكلي الذي يوفر لأصحاب الشبكات (ISPs) ومزودي الخدمة في العراق والوطن العربي تحكماً مطلقاً، سلساً، وآمناً بنسبة 100% في المشتركين، أجهزة الراوتر (MikroTik Core)، الكروت، الخصائص وعقد الاتصالات.

تدمج المنظومة بين قوة **محرر السكربتات المتقدم**، **المزامنة الحية عبر الـ REST API**، و**المساعد الذكي القائم على الذكاء الاصطناعي (Gemini Core)** لتوفير واجهة تفاعلية خالية من الأخطاء تضمن أعلى مستويات الاستقرار والأداء.

---

## 🎨 الهوية البصرية والتصميم (Visual Identity)
تعتمد المنظومة على تصميم عصري غامر **(Cosmic Slate Dark Theme)** مصمم خصيصاً لراحة أعين مدراء الشبكات الذين يقضون ساعات طويلة أمام الشاشات:
- **توزيع الفراغات الهندسية**: مسافات حرة وهوامش واسعة تمنع التشتت وتسهل القراءة السريعة للبيانات.
- **تكامل الخطوط**: دمج خط **Space Grotesk** الجريء للعناوين التكنولوجية مع خط **Inter** المرن للنصوص العادية، وخط **JetBrains Mono** الاحترافي لإظهار تفاصيل الاكواد والسكربتات وشاشات الطرفية (Terminal).
- **مؤشرات الحالات**: ألوان مدروسة بدقة وتأثيرات نبض حركية (Visual Pulse Interactions) توضح فورا حالة اتصال الراوتر وخطوط المشتركين (متصل، منتهي، معطل، جاري الفحص).

## ✨ المميزات الرئيسية للمنظومة (Key Features)

### 1️⃣ لوحة معلومات حية وتفاعلية (Dashboard Overview)
- **لوحة المراقبة السريعة**: توضح حالة استهلاك المعالج (CPU)، الذاكرة العشوائية (RAM)، عدد الاتصالات الحية لراوتر **RB-4011** الافتراضي.
- **التشخيص التلقائي الـ وعميق الـ (Diagnostic Tool)**: فحص متكامل للمنافذ، والـ REST API، ومعدل التأخير (Latency) مع تفصيل حي يظهر في شاشة الكونسول المرافقة.
- **فلترة السجلات المتقدمة (Systen & Mikrotik Logs)**: جلب السجلات وفلترتها لحظياً حسب تصنيفات واضحة: (عمليات مالية، مزامنة واتصالات، أخطاء وتحذيرات) لتسريع حل مشكلات الشبكة.
- **إحصائيات ذكية**: تمثيل بياني ديناميكي لتوزيع المشتركين حسب الأبراج والمحافظات وشعبية ملفات الاشتراكات (Active Profiles distribution).

### 2️⃣ نظام إدارة المشتركين ذو الفعالية القصوى (Subscriber Manager)
- **إدارة دورة حياة المشترك الكاملة**: (إضافة، تعديل الحساب، تجديد فوري، إيقاف مؤقت، تفعيل، حذف).
- **الاستيراد الجماعي السلس (Bulk Import)**: أداة خارقة لاستيراد آلاف المشتركين دفعة واحدة عبر سحب وإلقاء ملفات Excel/CSV أو اختيارها يدوياً مع فلترة وتصحيح العناوين والرموز تلقائياً.
- **سجلات المزامنة الحية (Live Rest API Sync Logs)**: واجهة لمراقبة الأوامر المرسلة إلى الـ Mikrotik بشكل مباشر بالرموز التعبيرية والألوان لبيان نجاح أو فشل المزامنة.
- **البحث والفلترة المتقاطعة**: البحث بالاسم، الهاتف، اسم المستخدم (PPPoE / Hotspot)، البرج، أو تصفية القائمة حسب حالة الاشتراك ومستواها المالي.

### 3️⃣ منشئ الكروت وذكاء الأكواد (Voucher & Coupon Generator)
- **توليد الكروت الفوري**: إمكانية توليد كروت عشوائية مشفرة ومعقدة (Random Secure PINs) مع تحديد البادئة وصلاحية الكارت والملف التعريفي (Profile) المرتبط به.
- **قوالب مخصصة للطباعة**: نظام طباعة متوافق مع الطابعات الحرارية لشبكات الإنترنت بضغطة زر واحدة.
- **تصدير سكربت توليد الكروت**: يقوم النظام بإنشاء وتجهيز السكربت البرمجي المقابل لنسخه ولصقه مباشرة داخل جهاز المايكروتك لإنشاء حسابات المستخدمين هناك.

### 4️⃣ إدارة السرع والملفات الراقية (Profiles Manager)
- **مرونة تامة في تحديد الخصائص**: حد سرعة التحميل والرفع (Queue Limits)، تحديد مدة الصلاحية لكل مشترك، والحد الأقصى للأجهزة المتصلة بالحساب الواحد تلقائياً.
- **مبدل العملات المتكامل**: إمكانية تحديد الأسعار بـ **الدينار العراقي (IQD)** أو **الدولار الأمريكي (USD)** مع تعديل ديناميكي فوري وموحد لأسعار الاشتراكات عبر اللوحة بالكامل.

### 5️⃣ محاكي الطرفية والسكربتات الذاتية (Script Terminal & Mikrotik Shell)
- **شاشة هيدروليكية كاملة**: محاكاة لوحدة التحكم (Terminal Emulator) لتشغيل أوامر مايكروتك وحقن السكربتات بأسلوب ملون واحترافي.
- **مكتبة السكربتات الجاهزة**: الوصول الفوري لأقوى السكربتات المخصصة لإدارة شبكات المايكروتك بنقرة واحدة لشحن الكروت، إيقاف المنتهي صلاحيتهم، تعطيل المشتركين وتحذيرهم عبر الويب.

### 6️⃣ المساعد بالذكاء الاصطناعي (AI Copilot Companion)
- مساعد ذكي ومبني برمجياً عبر محرك **Gemini Core** لمساعدة المهندسين ومدراء المنظومة لحساب تقسيمات الـ IP، وتوليد الشيفرات المعقدة لأوامر المايكروتك، وحل مشكلات التوجيه (Routing) وصيانة الشبكة بشكل فوري.

### 7️⃣ بوابة إدارة التنبيهات والبوت (Alert & Bot Center)
- تحكم مركزي بقنوات البث والتنبيه للمشتركين وصاحب المنظومة عن السيرفر وتكامل سهل لإعدادات الـ Webhooks لتلقي التنبيهات المباشرة على **تطبيق تليجرام (Telegram Bots)**.

## 📱 ميزة التثبيت الرائدة PWA والتوافق الكامل مع الـ VPN (PWA Support)

إن أهم إضافات الإصدار v4 هي الدعم المتكامل كـ **تطبيق ويب تقدمي (Progressive Web App)** مدعوم بملفات `manifest.json` و `sw.js` وخدمات الـ **Service Worker** المتقدمة:

- **مشكلة الـ VPN الشائعة لمهندسي وتجار الإنترنت**: عند فتح الـ VPN للوصول لأجهزة الراديو والمايكروتك بالآيبيات الافتراضية، غالباً ما يفقد المتصفح روابط الاتصال بالإنترنت الخارجي أو تتشوه إعدادات الجلسة وتتطلب تسجيل وبحث جديد.
- **الحل السحري لـ SuperSAS v4**: يمكن للمستخدم الآن تثبيت اللوحة اختيارياً كـ **تطبيق منفصل تماماً (Standalone Desktop / Mobile App)**.
- **على هواتف Android**: تظهر نافذة تثبيت اختيارية لتحويل المنظومة لأيقونة بالشاشة الرئيسية كأي تطبيق رسمي من متجر Play.
- **على أجهزة الـ iOS (iPhone / iPad)**: دليل إرشادي مدمج لإضافة اللوحة لشاشة الهاتف عبر متصفح Safari بضغطة زر واحدة.
- **على أجهزة الـ Windows & Mac**: يمكن تثبيت اللوحة لتعمل كبرنامج منفصل مستقل عن الـ Chrome أو Edge بنافذتها المستقلة دون شريط المتصفح العلوي.
- **الفائدة الفنية**: بمجرد تثبيت التطبيق وتفعيله كـ App مستقل، يتم عزل وتأمين الدخول واستخدام الذاكرة المؤقتة المحلية (Local App Cache Storage)، مما يحمي المنظومة من تداخل الـ VPN ويسهل فتحها وتحديث المحتوى المالي والبياني للمشتركين بشكل مستمر دون انقطاع.

## 🛠️ تفاصيل الهيكل البرمجي والملفات (Codebase Infrastructure)

يعتمد مشروع SuperSAS v4 على هيكلية معيارية لضمان الاستقرار التام وسهول الصيانة والسرعة الفائقة لخدمة أكثر من **200 مشترك ومزود خدمة محلي**:

```bash
├── public/
│   ├── icon.svg            # شعار السوبر عالي الدقة والمناسب لجميع أحجام الشاشات
│   ├── manifest.json       # ملف التهيئة والتخصيص لـ PWA للهواتف والكمبيوتر
│   └── sw.js               # Service Worker لإدارة شبكة الاتصال، وتحميل الأشرطة والـ Caching
├── src/
│   ├── components/         # المكونات البرمجية المجزأة لمنع بطء الترجمة والتحميل
│   │   ├── AiCopilot.tsx          # كونسول ومحاور الذكاء الاصطناعي (Gemini)
│   │   ├── AlertManager.tsx       # إدارة التنبيهات والويب هوك وتليجرام
│   │   ├── DashboardOverview.tsx  # لوحة التحكم ومحاكي التشخيص ومراقبة الراوتر والسجلات
│   │   ├── ProfileManager.tsx     # لوحة إعداد الخصائص والسرعات، وتحديد العملة الموحدة
│   │   ├── PwaInstallPrompt.tsx   # منبثق التثبيت الذكي مع شروحات الآيفون والاندرويد والويندوز
│   │   ├── RouterConfig.tsx       # إعدادات الرواترات والاتصال الافتراضي بالمايكروتك
│   │   ├── ScriptTerminal.tsx     # محاكي الكونسول الميكروتك ومكتبة السكريبتات الجاهزة
│   │   ├── SubscriberManager.tsx  # وحدة معالجة وتجديد واستيراد المشتركين الفائقة
│   │   └── VoucherManager.tsx     # توليد الكروت وأكواد شحن الرصيد والطباعة
│   ├── context/
│   │   └── SystemContext.tsx      # محرك إدارة البيانات والحالة المركزية (State Engine) وحفظ البيانات تلقائياً
│   ├── App.tsx             # الهيكل الأساسي للبرنامج، تقسيم الشاشة والقوائم الجانبية
│   ├── main.tsx            # ملف ربط التطبيق وتسجيل الـ Service Worker
│   └── index.css           # ملف التنسيق الكلي المعتمد على Tailwind CSS وتوزيع الخطوط الاحترافية
└── package.json            # إدارة حزم البرمجيات وأوامر التجميع والتشغيل والامان
```

## ⚙️ تشغيل وفحص المنظومة محلياً (Running Locally)

لتطوير المنظومة على خادمك الخاص، تأكد من تنصيب بيئة **Node.js**:

1. **تثبيت الحزم البرمجية**:
   ```bash
   npm install
   ```

2. **التشغيل ببيئة التطوير الحية (Dev Mode)**:
   ```bash
   npm run dev
   ```

3. **بناء المشروع وتجميعه للإنتاج (Production Build)**:
   ```bash
   npm run build
   ```
